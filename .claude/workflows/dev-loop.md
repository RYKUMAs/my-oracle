## Workflow: Autonomous Dev Loop

For any complex or technical task:

1. Architect:
   - Analyze the problem
   - Break it into clear steps
   - Define project structure first

2. Engineer:
   - Implement minimal working solution

3. Tester:
   - Create test cases
   - Validate logic and expected behavior

4. Debugger:
   - If failure → find root cause
   - Apply minimal fix

5. Repeat loop until:
   - Code works
   - Logic is correct

6. Final Verification:
   - Ensure it works in real environment
