# Retrospective Skill

## คำแนะนำ (Instructions)
- รวบรวมสรุปงานและสิ่งที่เรียนรู้จาก session
- บันทึกลง Memory
- แสดงสรุปให้ผู้ใช้

## ขั้นตอน (Steps)
1. โหลด session data ล่าสุดจาก Memory
2. วิเคราะห์ความสำเร็จ, ปัญหา, จุดที่ต้องปรับปรุง
3. บันทึก insight ลง Memory
4. แสดงรายงานสรุปให้ผู้ใช้
