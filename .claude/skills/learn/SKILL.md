# Learn Skill

## คำแนะนำ (Instructions)
- เรียก agent หลายตัวพร้อมกันเพื่อสำรวจ codebase
- วิเคราะห์ dependencies และตรวจสอบ bug
- บันทึก insight ลง Memory

## ขั้นตอน (Steps)
1. เลือก agent: architect, engineer, debugger
2. agent เริ่มสำรวจ codebase
3. รวบรวมข้อมูลจาก agent ทั้งหมด
4. วิเคราะห์ปัญหาและโอกาส
5. บันทึก insight ลง Memory
6. แสดงรายงานสรุป
