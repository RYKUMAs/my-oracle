# Trace Skill

## คำแนะนำ (Instructions)
- ค้นหาไฟล์หรือ project จาก git history
- ใช้สำหรับวิเคราะห์ dependencies หรือ bug

## ขั้นตอน (Steps)
1. ตรวจสอบ git repo และ history
2. ค้นหาไฟล์หรือ project ตาม keyword
3. แสดงผลการค้นหาให้ผู้ใช้
4. บันทึกการค้นหาใน Memory
