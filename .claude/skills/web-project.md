## Skill: Web Project Setup

## Steps
1. Define project structure
2. Create frontend folder
3. Create backend folder
4. Separate HTML, CSS, JS
5. Ensure clean architecture

## Structure
- /frontend
  - index.html
  - style.css
  - script.js
- /backend
  - app.py (or server.js)

## Rules
- No single-file project
- Must separate concerns
